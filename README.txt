Just Ratatoskr。（Windows）
素材文件夹：
Rat-鼠标滚轮控制的逐帧动画
Ato-光标替换文件
Clickleft-左键单击特效（图片与音效）
ClickRight-右键单击特效（图片与音效）

依赖项：
sys
os
random
winreg
PyQt5
pynput
keyboard
functools

【提醒】
-依赖项中包含全局钩子库，或许需要管理员权限来运行。
-启动后，键入Ctrl+Shift+Q快捷键组合退出。
-启用后，将会在您的Windows注册表内写入开机自启动项，若想彻底摆脱恼人的Ratatoskr而不必每次启动后按下Shift+Ctrl+Q杀虫剂，请双击运行提供的标准unistall脚本卸载。
-按下中键，操控Rat光标表演皮影戏——让它说些睿智的话语。
