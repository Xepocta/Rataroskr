Just Ratatoskr。（Windows）
【资源文件夹】
Rat-鼠标滚轮控制的逐帧动画
Ato-光标替换文件
click_left-左键单击特效（图片与音效）
click_right-右键单击特效（图片与音效）

按下中键，操控聪明的Rat表演皮影戏——让它带来一些有益于身心的睿智妙语——等等，那听起来有些熟悉？
【组合键控制】
-Ctrl+Shift+Alt+T 命令Ratatoskr陷入深思。
-Ctrl+Shift+Alt+H 隐匿Ratatoskr的踪迹。
-Ctrl+Shift+Alt+Q 关闭Ratatoskr。

-Ctrl+Shift+Alt+up 命令Ratatoskr大声说出头脑深处的秘密！
-Ctrl+Shift+Alt+down 命令Ratatoskr暂时停止说出头脑深处的秘密。
-Ctrl+Shift+Alt+left 命令Ratatoskr重复一遍刚才说过的秘密。
-Ctrl+Shift+Alt+right 命令Ratatoskr未卜先知将要说出的秘密。

-Ctrl+Alt+1+up 让它加快语速。
-Ctrl+Alt+1+down 让它放缓语速。
-Ctrl+Alt+2+up 让它提高讲话的音调。
-Ctrl+Alt+2+down 让它降低讲话的音调。
-Ctrl+Alt+3+up 让它大声一些。
-Ctrl+Alt+3+down 让它小声一些。
-Ctrl+Alt+4+up 让它增加字词间的间隔。
-Ctrl+Alt+4+down 让它降低字词间的间隔。
-Ctrl+Alt+5+up 提高念出大写拉丁字母的音调。
-Ctrl+Alt+5+down 降低念出小写拉丁字母的音调。
-Ctrl+Alt+6+up 切换上一种语言。
-Ctrl+Alt+6+down 切换下一种语言。
-Ctrl+Alt+7+up 切换上一种声音。
-Ctrl+Alt+7+down 切换下一种声音。
