Just Ratatoskr。（Windows）
素材文件夹：
Rat-鼠标滚轮控制的逐帧动画
Ato-光标替换文件
click_left-左键单击特效（图片与音效）
click_right-右键单击特效（图片与音效）

依赖项：
sys
os
random
PyQt5
pynput
keyboard
functools

【提醒】
-依赖项中包含全局钩子库，或许需要管理员权限来运行。
-按下中键，操控聪明的Rat表演皮影戏——让它带来一些有益于身心的睿智妙语——等等，那听起来有些熟悉？
-启动后，键入Ctrl+Shift+E快捷键组合打开Ratatoskr的对话框。
-启动后，键入Ctrl+Shift+W快捷键组合隐匿Ratatoskr的踪迹。
-启动后，键入Ctrl+Shift+Q快捷键组合关闭Ratatoskr。
-右键并以管理员身份运行"morning.bat"，将会在您的Windows注册表内写入开机自启动项，若想彻底摆脱恼人的Ratatoskr而不必每次启动后按下Shift+Ctrl+Q杀虫剂，请双击运行提供的标准unistall脚本完全卸载它。（如果您想留下其中的美术或音效素材，千万记得备份）
